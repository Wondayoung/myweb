코딩

코딩은 기능을 구현하는 것을 넘어, 사용자 경험과 목적을 명확히 반영하는 설계의 연장선이라고 생각합니다. 저는 디자인 감각을 바탕으로 직관적이고 일관된 UI/UX를 구현하며, 프론트엔드 영역을 중심으로 다양한 프로젝트를 경험해왔습니다.

사용 기술 및 스킬셋:

  • HTML/CSS – 반응형 웹페이지 및 포트폴리오 사이트 구축 경험
  • JavaScript – 기본적인 DOM 조작, 이벤트 처리, 슬라이드/인터랙션 구현
  • Bootstrap – 효율적인 레이아웃 설계와 디자인 시스템 활용
  • Figma → 코드 전환 – 디자인 시안에 맞춘 정확한 구현 경험

작업물은 사용자의 흐름을 고려한 구조적 마크업을 지향하며, 유지보수가 쉬운 깔끔한 코드 작성을 중시합니다. 또한 디자인 감성을 살린 애니메이션과 전환 효과로 사용자 경험을 풍부하게 만듭니다.
